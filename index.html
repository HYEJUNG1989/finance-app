🔧 Firebase 연결 설정

아래 단계대로 하면 5분 안에 폰↔PC 동기화가 됩니다. 모두 무료예요!

1
Firebase 프로젝트 만들기console.firebase.google.com → "프로젝트 추가" → 이름 입력 → 애널리틱스 비활성화 → 완료
2
Realtime Database 생성왼쪽 "빌드" → "Realtime Database" → "데이터베이스 만들기" → 테스트 모드 → 완료
3
앱 등록 & 설정 복사프로젝트 개요 → 웹(/>) 클릭 → 닉네임 입력 → firebaseConfig 값들을 아래에 입력
