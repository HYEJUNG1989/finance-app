🔧 Firebase 연결 설정

아래 단계대로 하면 5분 안에 폰↔PC 동기화가 됩니다.
모두 무료예요!

1
Firebase 프로젝트 만들기 console.firebase.google.com 접속 → "프로젝트 추가" → 이름 입력 → Google 애널리틱스 비활성화 → 완료
2
Realtime Database 생성 왼쪽 메뉴 "빌드" → "Realtime Database" → "데이터베이스 만들기" → 테스트 모드 선택 → 완료
3
앱 등록 & 설정 복사 프로젝트 개요 → 웹() 클릭 → 앱 닉네임 입력 → firebaseConfig 의 각 값을 아래에 붙여넣기
